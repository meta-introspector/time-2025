# Pure Functions

In computer programming, a pure function is a function that has two main characteristics:
1.  **Deterministic:** Given the same input, it will always return the same output.
2.  **No Side Effects:** It does not cause any observable side effects outside its local environment (e.g., it doesn't modify global variables, perform I/O, or change the state of objects passed by reference). Pure functions are a cornerstone of functional programming, contributing to more predictable and testable code.
