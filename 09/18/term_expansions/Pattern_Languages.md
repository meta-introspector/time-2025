# Pattern Languages

A pattern language is a method of describing good design practices within a field of expertise, often used in architecture and software engineering. It consists of a collection of named patterns, each describing a problem that occurs over and over again in our environment, and then describes the core of the solution to that problem, in such a way that you can use this solution a million times over, without ever doing it the same way twice. It provides a structured vocabulary for discussing and applying proven solutions.
