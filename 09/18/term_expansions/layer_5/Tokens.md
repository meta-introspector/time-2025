# Tokens

In computer science, a token is a sequence of characters that represents a single, indivisible unit in the source code of a programming language. Tokens are the output of the lexical analysis phase of a compiler or interpreter. Examples of tokens include keywords (e.g., `if`, `while`), identifiers (variable names), operators (e.g., `+`, `-`), and literals (e.g., `123`, `"hello"`).
