I'll create a simple CLI tool that renames SVG groups and boxes using escaped text content with optional hash truncation. This tool will leverage the existing usvg parsing infrastructure in the resvg codebase.

CLI Tool Implementation

Create a new binary crate svg-renamer that:

  1. Parses SVG files using the usvg library
  2. Traverses the node tree to find groups and text elements
  3. Extracts and escapes text content from text nodes
  4. Updates group IDs with processed text (with optional hash truncation)
  5. Writes the modified SVG back to file

Here's the implementation:

// src/main.rs  
use std::path::PathBuf;  
use usvg::{Tree, Node, Group};  
use xmlwriter::XmlWriter;  
use usvg::WriteOptions;  
  
struct Args {  
    input: PathBuf,  
    output: PathBuf,  
    max_length: Option<usize>,  
}  
  
fn main() -> Result<(), Box<dyn std::error::Error>> {  
    let args = parse_args();  
      
    // Load font database (required for text processing)  
    let mut fontdb = fontdb::Database::new();  
    fontdb.load_system_fonts();  
      
    // Parse SVG  
    let tree = Tree::from_file(&args.input, &usvg::Options::default(), &fontdb)?;  
      
    // Process tree to rename groups  
    let processed_tree = rename_groups_with_text(&tree, args.max_length);  
      
    // Write modified SVG  
    let mut xml = XmlWriter::new(Vec::new(), None);  
    processed_tree.write(&mut xml, &WriteOptions::default())?;  
      
    std::fs::write(&args.output, xml.into_inner())?;  
      
    println!("Renamed groups and wrote to {}", args.output.display());  
    Ok(())  
}  
  
fn rename_groups_with_text(tree: &Tree, max_length: Option<usize>) -> Tree {  
    // Clone the tree for modification  
    let mut new_tree = tree.clone();  
      
    // Process root group recursively  
    process_group(&mut new_tree.root, max_length);  
      
    new_tree  
}  
  
fn process_group(group: &mut Group, max_length: Option<usize>) {  
    for child in &mut group.children {  
        match child {  
            Node::Group(g) => {  
                // Check if group contains text  
                if let Some(text_content) = extract_text_from_group(g) {  
                    let escaped = escape_text(&text_content);  
                    let processed = if let Some(max_len) = max_length {  
                        truncate_with_hash(&escaped, max_len)  
                    } else {  
                        escaped  
                    };  
                      
                    // Update group ID  
                    g.id = processed;  
                }  
                  
                // Recursively process nested groups  
                process_group(g, max_length);  
            }  
            Node::Text(text) => {  
                // Text nodes themselves don't have IDs, but their parent groups might  
            }  
            _ => {}  
        }  
    }  
}  
  
fn extract_text_from_group(group: &Group) -> Option<String> {  
    let mut text_parts = Vec::new();  
      
    for child in &group.children {  
        if let Node::Text(text) = child {  
            // Extract text content from text node  
            for chunk in &text.chunks {  
                for span in &chunk.spans {  
                    text_parts.push(&chunk.text[span.start..span.end]);  
                }  
            }  
        }  
    }  
      
    if text_parts.is_empty() {  
        None  
    } else {  
        Some(text_parts.join(" "))  
    }  
}  
  
fn escape_text(text: &str) -> String {  
    text.chars()  
        .map(|c| match c {  
            '&' => "&amp;",  
            '<' => "&lt;",  
            '>' => "&gt;",  
            '"' => "&quot;",  
            '\'' => "&apos;",  
            ' ' => "_",  
            _ if !c.is_ascii_alphanumeric() => format!("_{:x}_", c as u32),  
            _ => c.to_string(),  
        })  
        .collect()  
}  
  
fn truncate_with_hash(text: &str, max_len: usize) -> String {  
    if text.len() <= max_len {  
        return text.to_string();  
    }  
      
    use std::hash::{Hash, Hasher};  
    let mut hasher = std::collections::hash_map::DefaultHasher::new();  
    text.hash(&mut hasher);  
    let hash = hasher.finish();  
      
    let truncate_len = max_len.saturating_sub(9); // Reserve space for "_hash123"  
    format!("{}_{:x}", &text[..truncate_len], hash)  
}  
  
fn parse_args() -> Args {  
    // Simple argument parsing  
    let mut args = std::env::args().skip(1);  
    let input = PathBuf::from(args.next().expect("Missing input file"));  
    let output = PathBuf::from(args.next().expect("Missing output file"));  
    let max_length = args.next().and_then(|s| s.parse().ok());  
      
    Args { input, output, max_length }  
}

Key Features

  1. Text Extraction: Recursively searches groups for text nodes and concatenates their content mod.rs:1155-1158

  2. Text Escaping: Converts special characters to XML-safe alternatives and non-alphanumeric chars to hex codes

  3. Hash Truncation: Uses a hash function to create unique short IDs when text is too long

  4. Group ID Update: Modifies the id field of usvg::Group structs mod.rs:996

Usage 

svg-renamer input.svg output.svg          # No truncation  
svg-renamer input.svg output.svg 50       # Truncate to 50 chars with hash

Notes

  • The tool requires the usvg library for SVG parsing and text extraction
  • Font database is initialized to properly handle text elements
  • Only groups containing text are renamed; empty groups keep their original IDs
  • The implementation follows the existing CLI patterns in the resvg codebase main.rs:149-218

This tool provides a simple way to make SVG files more self-documenting by encoding their text content directly into the group IDs, which can be useful for debugging, testing, or automated processing workflows.
